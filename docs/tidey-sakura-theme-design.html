設計原則

1. 白紙に桜 — 主內容區用胡粉（近白），如和紙底色。桜粉只出現在 sidebar，如落花點綴紙邊。
2. 墨と花 — 文字用憲法（深棕），不用純黑。毛筆墨色與花朵的溫潤對比。對比度 10.8:1 超過 WCAG AAA。
3. 一輪の紅 — Focus indicator 和游標都用今様（深粉紅），整個畫面最飽和的色。對比度 4.2:1 達 AA。
4. 春の彩り — ANSI 16 色全部對應到和色，讓終端輸出也有春天的氣質。桔梗紫、若竹綠、山吹金都是花園裡的色彩。
5. hover = 今様@6% — Hover 態統一用今様極淡透明疊加。dark theme 用白色疊加，light theme 用 accent 疊加，各自協調。

Why Tidey

Agent-Aware Workspaces

Each workspace is an agent session with its own terminal, status, and notifications. Tidey tracks what your agent is doing — Running, Idle, or Needs Input — so you always know at a glance. ⌘1–⌘9 to switch instantly.

Code Editor, Built In

⌘-click any file path in your terminal to open it. Syntax highlighting, search, save. Auto-refreshes when files change externally — you always see the latest version of what your agent wrote. Powered by Monaco.

Terminal-First, Not IDE-First

Built on iTerm2's battle-tested terminal emulation. Sidebar, editor, and file tree are all independently collapsible. Show what you need, hide what you don't. Works with tmux. Your terminal is the center.

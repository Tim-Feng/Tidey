tidey

Terminal-first IDE for AI agents. One window, every context.

Download for macOS macOS 12+ · Apple Silicon & Intel · GPLv2+
[ screenshot ]
01

Agent-Aware Workspaces

Each workspace is an agent session with its own terminal, status, and notifications. Tidey tracks Running, Idle, and Needs Input automatically. ⌘1–⌘9 to switch.

02

Code Editor, Built In

⌘-click any file path in your terminal to open it — right next to where your agent is working. Syntax highlighting, search, save. Powered by Monaco.

03

Terminal-First

Built on iTerm2. Sidebar, editor, file tree — all collapsible. Your terminal is the center. Everything else supports it. Works with tmux.
